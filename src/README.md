# A-Simple-File-System


Before running any of the programs, run the "make" command to compile

Part I:
To invoke the disk info program,

./diskinfo <disk>.ima

----------------------------
Part II:
To invoke the disk list program

./disklist <disk>.ima

----------------------------
Part III:
To invoke the disk get program

./diskget <disk>.ima <file>

NOTE: When entering the file name, use capitals for both the file name and the file extension

----------------------------
Part IV:
To invoke the disk put program

./diskput <disk>.ima [directory path]<file>
